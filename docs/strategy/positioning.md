# Positioning

## One-Line Positioning

**Aun Stack is an agent-agnostic control plane for enterprise AI workflows.**

It lets teams swap AI agents while preserving context, permissions, decisions, handoffs, and audit trails.

## Core Phrase

**Swap agents, not workflows.**

Japanese:

**エージェントは替えられる。業務フローは壊さない。**

## What Aun Stack Is

Aun Stack is the operational layer around AI agents.

It is designed for teams that use or plan to use:

- Codex
- Claude Code
- Cursor
- Gemini
- Copilot
- local LLM agents
- custom MCP agents
- future AI agents

Aun Stack keeps the operational layer stable even when the underlying agent changes.

## What Aun Stack Is Not

Aun Stack is not:

- a single MCP server
- a replacement for Codex or Claude Code
- a simple chat room for agents
- a generic ChatGPT training product
- an all-purpose automation promise
- a fully autonomous AI system without human control

## Key Differentiation

### 1. Agent-agnostic operation
Use the best agent for each job without rebuilding the workflow.

### 2. PR continuity
Keep PRs small and reviewable while preserving feature-level context across multiple PRs.

### 3. Human-in-the-loop control
Important actions should be approved, recorded, and auditable.

### 4. Decision memory
Decisions, assumptions, constraints, and human intent should not disappear between sessions or agents.

### 5. Enterprise readiness
Permissions, audit logs, approvals, and workflow visibility are first-class requirements.

## Message by Customer Type

### AI implementation / DX companies
You already have customers asking about AI. Aun Stack helps you review AI project risks, design safer implementations, and provide enterprise-grade AgentOps thinking.

### Development companies
AI coding agents can speed up implementation, but teams need small PRs, reviewable changes, and continuity across agents. Aun Stack helps preserve context and decisions across the workflow.

### R&D companies
Research, partner communication, documents, and internal knowledge can be AI-assisted without touching regulated clinical judgment. Aun Stack supports safe AI operational foundations.

### Field-industry companies
AI can reduce reporting, handoff, estimate, and communication work, but human approval and operational history must remain clear.

## Product Theme

**AI agents are getting powerful. Enterprises need control.**

Japanese:

**AIエージェントは強くなる。だからこそ、企業には“任せ方”の基盤が必要になる。**

## First Public Narrative

AI coding agents can already write code. Personal agents can already operate tools. The problem is no longer whether AI can do tasks.

The problem is whether companies can safely manage who asked the agent, what context it used, what decisions it followed, what permissions it had, what it changed, and how the next agent or human continues the work.

Aun Stack exists to solve that operational gap.
